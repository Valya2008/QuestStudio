# QuestStudio
Мобильное приложение для автоматизации бронирования квестов в квест-студии.  

<img width="400" height="400" alt="image" src="https://github.com/user-attachments/assets/67fc494c-a846-44bc-a0c6-baca8e27ebef" />

## О проекте
Автоматизация работы квест-студии – это мобильное приложение, созданное для упрощения взаимодействия клиентов с квест-студией. Платформа позволяет клиентам просматривать каталог квест-сценариев и актёров, бронировать квесты с помощью интерактивного календаря с цветовой индикацией стоимости, оставлять отзывы, а администратору — управлять контентом и отслеживать статистику.  

## Основные возможности
* регистрация и авторизация пользователей
*	просмотр каталога квест-сценариев
*	просмотр списка актеров
*	бронирование квест-сценариев
*	бронирование квест-сценариев
*	просмотр истории посещений клиента
*	отмена записи
*	написание и просмотр отзывов
*	управление	каталогом квест-сценариев со стороны администратора
*	управление списком актеров со стороны администратора
*	ведение бронирований со стороны администратора

## Установка и запуск
### Требования
* Android Studio (последняя версия)  

* JDK 11 или выше  

* Устройство или эмулятор с Android 8.0+  

* Аккаунт Firebase

## Структура проекта  
```
app/src/main/java/com/example/queststudio/
│  
├── activities/  
│   ├── LoginActivity.java              # Вход в систему  
│   ├── RegistrationActivity.java       # Регистрация  
│   ├── MainActivity.java               # Главный экран  
│   ├── ClientMenuActivity.java         # Меню клиента  
│   ├── AdminMenuActivity.java          # Меню администратора  
│   ├── CategoriesActivity.java         # Список категорий квестов  
│   ├── QuestsActivity.java             # Список квестов  
│   ├── QuestDetailActivity.java        # Детальная информация о квесте  
│   ├── BookingActivity.java            # Бронирование  
│   ├── MyBookingsActivity.java         # Мои бронирования  
│   ├── ReviewsActivity.java            # Отзывы  
│   ├── ActorsActivity.java             # Список актёров  
│   ├── AddQuestActivity.java           # Добавление квеста (админ)  
│   ├── AddActorActivity.java           # Добавление актёра (админ)  
│   ├── ManageBookingsActivity.java     # Управление бронированиями (админ)  
│   ├── ManageReviewsActivity.java      # Управление отзывами (админ)  
│  
├── adapters/  
│   ├── CategoryAdapter.java            # Адаптер для категорий  
│   ├── QuestAdapter.java               # Адаптер для квестов  
│   ├── SessionAdapter.java             # Адаптер для сеансов  
│   ├── ActorAdapter.java               # Адаптер для актёров  
│   ├── BookingAdapter.java             # Адаптер для бронирований  
│   └── ReviewAdapter.java              # Адаптер для отзывов  
│  
└── model/  
    ├── User.java                       # Модель пользователя  
    ├── Category.java                   # Модель категории  
    ├── Quest.java                      # Модель квеста  
    ├── Actor.java                      # Модель актёра  
    ├── Booking.java                    # Модель бронирования  
    └── Review.java                     # Модель отзыва  
```
